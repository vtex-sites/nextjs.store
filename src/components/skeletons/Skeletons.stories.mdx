# Skeletons

This feature improves the store's experience and perceived performance through loading placeholders.

</header>

## Overview

<code>Skeletons</code> are like an element's or section's wireframe. The wireframe
is a placeholder for these elements/sections until the page finally loads.

Instead of showing a blank page, displaying <code>Skeletons</code> while loading content makes the user feel like the store is more responsive and faster.

Also by using <code>Skeletons</code> the store will be less likely to experience [Cumulative Layout Shilf (or CLS)](https://web.dev/i18n/en/cls/).
